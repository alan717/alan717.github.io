boot文件

boot

包括了kernel和ramdisk，参考/system/core/mkboot.img
bootloader通过mkbootimg长生的bootimg.h文件，更新kernel header包括了位置和ramdisk 大小在flash中，kernel的基地址，和命令行参数等。

kernel

kernel标准格式zImage、Image、Image.gz和ramdisk镜像一起刷入boot分区，从内存中启动。后边连着DTB

ramdisk

包括root file system mount as a rootfs.和kernel一起被mkbootfs 做成了boot flash到boot分区.

file system

文件系统有system、userdata、recovery.image. must use Yaffs2 format or sparse image format.

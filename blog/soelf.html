1.3.1. 页对齐

PAGE_START 这里4k对齐的实现是这样:

#define PAGE_SIZE 0x1000
#define PAGE_MASK (~(PAGE_SIZE-1))
#define PAGE_START(x) ((x) & PAGE_MASK)
这里就很简单描述就是保留高位，将低12位置为0. 对其成1000 4k.

计算结束为：

#define PAGE_END(x) PAGE_START((x) + (PAGE_SIZE-1))

这里讲对地址计算结束4k对齐

1.3.2. Weak symbol

默认情况下，在没有任何注解的情况下，对象文件中的符号是强的。在链接期间，强符号可以覆盖同名的弱符号。相比之下，在两个具有相同名称的强符号存在时，连接器将根据找到的第一个符号来解析该符号。这种行为允许可执行文件覆盖标准库函数，如malloc(3)。当链接二进制可执行文件时，弱声明符号不需要定义。相比之下，(默认情况下)声明的没有定义的强符号会触发未定义的符号链接错误。

1.3.2.1. 使用

// function declaration int __attribute__((weak)) power2(int x); // or int power2(int x) __attribute__((weak)); // variable declaration; extern int __attribute__((weak)) global_var;

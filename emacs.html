emacs使用记录

published on 2024-01-10 · tagged with #emacs

env

ubuntu 环境下使用emacs. emacs 使用快捷键之后就是肌肉记忆，不用记忆都记住了。很不自觉就使用上了。

foundemental

复制

选择使用shift 然后选择，然后 alt-w 复制，然后使用C-y粘贴。

保存

c-x c-s 保存。

另存为

C-x C-w 按提示，输入文件名，另存为.

打开文件

C—x C-f 打开。

退出

C-x C-c

配置

关闭备份文件 disable tlide file creation.

(setq make-backup-files nil) ; stop creating ~ files

use menu bar in terminal mode

F10 或者c-h k查看帮助.

取消 cancel

C-g or ESC ESC ESC tap three times.
